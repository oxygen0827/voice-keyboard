# Voice Keyboard Engine boundary

Voice Keyboard Engine is the local context that turns speech into text or text operations in the user's current input environment. It owns capture paths, dictation, instruction-driven text operations, tracked text safety, shortcut invocation, and reusable text memory, but it does not own TypeUp account UI, subscriptions, payments, cloud entitlement policy, provider billing, or hardware product lifecycle. This boundary keeps the repository useful as a standalone local engine while still allowing TypeUp desktop, TypeUp backend, hardware devices, and speech interpretation providers to integrate with it through explicit adapters.
